# WriteUp-Generator-Using-Llama-2

A simple and custom LLM to generate Cold mails or Cover Letters for a given set of roles. 
 
 Model used:
 LLaMa-2

 Frameworks: 
 - Langchain
 - Streamlit (Front-end)
